# Node Hierarchy Processor
A full stack app built for SRM and Bajaj's Full Stack Engineering Challenge.

## What it does
Takes graph edges like `A->B, A->C` and returns:
- Tree hierarchies with depth
- Cycle detection using DFS
- Invalid entry filtering
- Duplicate edge tracking

## Tech Stack
- Node.js + Express (backend)
- Vanilla HTML/CSS/JS (frontend)

## Run locally
npm install
node index.js
